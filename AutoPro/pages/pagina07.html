Sobre a AutoPro Oficina

A AutoPro é uma oficina mecânica fictícia criada para fins acadêmicos, como tema do Projeto Final da disciplina. O conteúdo do site como textos, serviços, preços e imagens foram criados exclusivamente para demonstrar os recursos aprendidos em sala de aula.

A oficina está ambientada em Belo Horizonte / MG e os dados de contato, endereço e valores são fictícios.

Recursos Utilizados no Projeto

HTML:

  1. Estrutura semântica: header, nav, main, section, footer
  2. Tabelas: table, tr, th, td, thead, tbody
  3. Formulários: form (method GET), input, select, textarea, label
  4. Mapa clicável: img usemap, map, area (rect e circle)
  5. Google Maps: iframe embed
  6. Âncoras: id + href="#id"
  7. Listas: ul, ol, li
  8. Links: target="_blank", rel="nofollow"
  9. Figure e img responsiva
  10. Entidades HTML: ©, ↑, ✓ etc.

CSS:

  1. Reset universal com seletor *
  2. Seletor html com font-size: 62.5% (base do rem)
  3. Unidade rem para fontes
  4. Bordas (border) e background-color
  5. position: fixed para o header
  6. display: flex no menu de navegação
  7. display: grid com grid-template-columns e gap
  8. Unidade fr (fragment)
  9. Função repeat() no grid
  10. Hover e transition (all 400ms ease-in-out)
  11. clamp() para fontes responsivas
  12. @media para responsividade
  13. rgba (transparência no header)
  14. text-shadow nos títulos
  15. nth-child para linhas alternadas da tabela

JavaScript:

  1. function (funções)
  2. var (variáveis)
  3. alert() com caracteres de escape (\n, \")
  4. document.getElementById()
  5. new Date() e toLocaleDateString()
  6. parseFloat() para cálculos
  7. Operações aritméticas (+, *, toFixed)
  8. innerHTML para exibir resultados
